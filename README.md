## Description

A simple task management application using [Nest](https://github.com/nestjs/nest) framework, [PostgreSQL](https://www.postgresql.org/docs/) and [Redis](https://redis.io/documentation) with e2e & unit test cases.

## Requirements

```bash
Docker and Docker Compose
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development mode
$ yarn start:dev

# development tools
$ yarn start:tools

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
